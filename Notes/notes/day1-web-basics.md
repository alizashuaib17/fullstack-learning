Day 1 - Web Basics

When a URL is typed into a browser and Enter is pressed, the browser first checks the DNS to find the IP address of the website. After getting the IP address, the browser sends an HTTP or HTTPS request to the web server. The server receives the request, processes it, and sends back the required files such as HTML, CSS, and JavaScript. The browser then downloads these files and renders them. Finally, the web page is displayed on the screen for the user to interact with.
